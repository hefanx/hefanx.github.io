Linux常用命令和快捷方式

发表于2023-02-18|更新于2023-02-21
|阅读量:

Linux复制粘贴快捷键

  • 复制快捷键：Ctrl+Insert
  • 粘贴快捷键：Shift+Insert

linux常用基本命令

  1. is 或list 作用：查看当前文件夹下的内容

  2. pwd 或print work directory 作用：查看当前所有文件夹

  3. cd[目录名] 或changge directory 作用：切换文件夹

  4. touch[文件名] 或touch 作用：如果文件不存在，新建文件

  5. mkdir[目录名] 或make directory 作用：新建指定文件夹

  6. rm[文件名] 或remove 作用：删除指定文件

  7. clear 作用：清屏

  8. find -name “公钥” 作用：搜索包含公钥的文件

  9. pwd 作用：查看当前工作路径

  10. cd ~ 作用：当前用户目录

  11. cd / 作用：返回到根目录

  12. Debian系统放行端口首先需要安装iptables，执行下面的命令：
    apt-get update

    apt-get install iptables

    例如放行443端口：iptables -I INPUT -p tcp –dport 443 -j ACCEPT

    iptables -A OUTPUT -p tcp –sport 443 -j ACCEPT

    保存放行规则：iptables-save

    查看已经放行的端口：/sbin/iptables -L -n

    检查端口是否放行的命令：iptables -S
